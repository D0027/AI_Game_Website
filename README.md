## ğŸ–¥ï¸ Mission Brief
**Neural Arena** is a comprehensive **AI Simulation Hub**. It demonstrates the practical application of complex algorithms through interactive gameplay. Unlike standard web games, every opponent here is driven by a specific Python-based AI agent running on the backend.

Built with a **Python Flask** backend and a reactive **Glassmorphism** frontend, the system simulates a high-tech command deck complete with terminal logs, ambient audio, and a responsive neon grid.

## ğŸ§  Algorithmic Engine

This project visualizes complex theoretical concepts through gameplay:

| Algorithm | Used In | Function |
| :--- | :--- | :--- |
| **Minimax** | Tic-Tac-Toe | Calculates all possible future moves to ensure a draw or win. |
| **A* (A-Star)** | Pac-Man / Slide Puzzle | Finds the absolute shortest path between two points. |
| **Backtracking** | Sudoku / N-Queens | Recursively solves constraint satisfaction problems. |
| **Genetic Algo** | Neon Runner | Evolves better agents over generations (survival of the fittest). |
| **BFS / DFS** | Snake / Maze | Breadth-First and Depth-First Search for graph traversal. |
| **Simulated Annealing** | Network (TSP) | Solves optimization problems by mimicking metallurgy. |
